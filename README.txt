This project is a automated Competition Maker in Java.
It has a Server side that saves the data locally on its end via JSON text files.

The project consists of 32 static teams that are made up.

In the project you can create fake contests from 4 16 or 32 teams, randomally, and have their outcomes.

You may display data of a team, either by its winning state, its name, or partial information in its summary.
